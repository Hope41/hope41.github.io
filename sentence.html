In this lesson, we will learn how to make a sentence generator. First we will discuss how it works, and then we can go on to JavaScript code.
To make our sentence, we need a few lists of different types of words:

articles
living nouns
dead nouns
verbs
adverbs
prepositions

Here is a simplified example of what each of those lists might contain:

articles:
A, The, One

living nouns:
Cat, Dog, Mouse

dead nouns:
Hill, Tree, Table

verbs:
Jumped, Walked, Ran

adverbs:
Quietly, Quickly, Sadly

prepositions:
next to, beside, against

Using this list, we can make a sentence.
We need to pick random words from each one in a certain order.
In this case the order should be:
article +
 living noun +
 verb +
 adverb +
 preposition +
 article +
 dead noun = sentence
This way, We could make a sentence like, the cat walked sadly next to one hill.

Like explained previously, We need to add each word together in a certain order.

In the combination function, all the abbreviated lists we made are changed to a random word from their list:

a = A random word in the articles list
l = A random word in the living_nouns list
d = A random word in the dead_nouns list
v = A random word in the verbs list
ad = A random word in the adverbs list
p = A random word in the prepositions list

 So, we can add ech of those to the str we made and finally add them to our textarea box 
            function press() {
                combination()
                str = ''
                
                str += a + ' '
                str += l
                if (a == 'some' || a == 'a few' || a == 'many') {
                    str += 's' // if the article is a plural, add an S on the end of the living noun
                }

                str += ' ' // remember to add spaces to your sentence!
                str += v + ' '
                str += ad + ' '
                str += p + ' '
                combination() // call up the combination again so the sentence may not have repeated articles: 'THE cat walked sadly on THE hill' or 'THE cat walked sadly on A FEW hills'
                str += a + ' '
                str += d
                if (a == 'some' || a == 'a few' || a == 'many') {
                    str += 's' // if the article is a plural, add an S on the end of the dead noun
                }
                str += '.' // add a full stop!
            }
