In this lesson, we will learn how to make a sentence generator. First we will discuss how it works, and then we can go on to JavaScript code.
To make our sentence, we need a few lists of different types of words:

articles
living nouns
dead nouns
verbs
adverbs
prepositions

Here is a simplified example of what each of those lists might contain:

articles:
A, The, One

living nouns:
Cat, Dog, Mouse

dead nouns:
Hill, Tree, Table

verbs:
Jumped, Walked, Ran

adverbs:
Quietly, Quickly, Sadly

prepositions:
next to, beside, against

Using this list, we can make a sentence.
We need to pick random words from each one in a certain order.
In this case the order should be:
article +
 living noun +
 verb +
 adverb +
 preposition +
 article +
 dead noun = sentence
This way, We could make a scentance like, the cat walked sadly next to one hill.
As the concept goes, that's it.
