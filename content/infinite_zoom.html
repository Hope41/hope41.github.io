Infinite Zoom

If you're using a mouse or touchpad, point in the direction you want to go. On a mobile device, tap on the place where you want to zoom in.

This experiment was created around the end of March 2022.
Here is the complete source code:

I have always wanted to make an infinite zoom because I enjoy the limitless exploration factor.

I was mainly inspired from Pixel Words by frank force, but wanted to make a JavaScript version. So, after many months of thinking and two-and-a-half days of developing, I got a result I was happy with.

This is how it works:

 1. Make a shape that consists of various squares put together.
 2. Put the shape on the screen.
 3. Zoom in.
 4. Once the squares making up the shape are bigger than a specified amount:
  1. Create another shape inside each of the old shapes' squares.
  2. Fade and then remove the old shapes' squares.
 5. Keep zooming in.

An annoying thing regarding the JavaScript fillRect() method is it starts to get slow when drawing over ~1000 rectangles on the screen. Apparently, both PutImageData() and webgl are super fast when drawing rectangles on the screen, but, they're a bit unfamiliar for me and I am going to stick with the slow technique for now.

Here it is!

The people who don't bother to read pages all the way to the bottom should now know better. That famous saying has been proved true once more, 'the cream sinks to the bottom'.

If you can be bothered, please read on below the improved example to understand how it works.

In the end I didn't use putImageData() or Webgl. Instead, I made a new canvas for the icon which was then called up whenever I wanted to draw one. This is a much faster technique and I therefore was able to generate icons from a greater distance without the performance suffering on most devices. These are the steps the program takes:

 1. Make a canvas that fits the screen.
 2. Make another canvas whose width and height is the same as the icon's width and height (i.e. 6x6).
 3. Call up the canvas using drawImage() when ever you want to use it.

The rest of the steps are covered in the previous list.
